# Contact Information Section Design Brief

## Overview
This design brief outlines the detailed design elements, interactions, and guidelines for the Contact Information section of the BKF Pharma website. The section aims to provide clear and accessible contact details for potential partners, investors, medical professionals, and the general public interested in BKF Pharma's work.

## Layout
### Structure
1. **Section Header**
   - Title: "Get in Touch"
   - Subtitle: "We would love to hear from you. Reach out with any inquiries or questions."

2. **Contact Information**
   - **Email Addresses**
     - General Inquiries: info@bkfpharma.com
     - Partner Relations: partners@bkfpharma.com
   - **Phone Numbers**
     - Main Office: +1 (800) 123-4567
     - Partner Relations: +1 (800) 234-5678
   - **Physical Address**
     - BKF Pharma Inc.
     - 123 Innovation Drive
     - Science City, CA 94000

3. **Contact Form**
   - Fields: Name, Email, Subject, Message
   - Submit Button: "Send Message"

4. **Map**
   - Interactive map showing the location of BKF Pharma’s main office.

## Color Scheme
- **Primary Colors:**
  - **Deep Blue (#003366):** Background for the section.
  - **White (#FFFFFF):** Text color and input field background.
- **Secondary Colors:**
  - **Teal (#008080):** Borders and highlights.
  - **Light Gray (#F2F2F2):** Input field borders and background for hovered items.
- **Accent Colors:**
  - **Bright Green (#00FF00):** Submit button background.

## Typography
- **Primary Font:** Helvetica Neue, sans-serif
  - **Headings:** Bold, uppercase
  - **Body Text:** Regular weight
  - **Input Fields:** Regular weight
  - **Submit Button:** Bold, uppercase

## Iconography
- **Email Icon:** Simple envelope icon
- **Phone Icon:** Simple phone icon
- **Location Icon:** Simple map pin icon
- **Form Icons:** Simple icons for Name, Email, Subject, and Message fields

## User Interface Components
### Buttons
- **Submit Button:**
  - Background Color: Bright Green (#00FF00)
  - Text Color: White (#FFFFFF)
  - Border Radius: 5px
  - Padding: 10px 20px
  - Hover Effect: Slight darkening of the button color

### Input Fields
- **Name, Email, Subject, Message:**
  - Border: 1px solid Light Gray (#F2F2F2)
  - Border Radius: 5px
  - Padding: 10px
  - Focus Effect: Border color changes to Teal (#008080)

### Contact Information Cards
- **Background Color:** White (#FFFFFF)
- **Border:** 1px solid Light Gray (#F2F2F2)
- **Border Radius:** 10px
- **Padding:** 20px

## Micro Interactions and Hover Effects
- **Hover Effects:**
  - Buttons: Slight darkening of the background color
  - Input Fields: Border color changes to Teal (#008080)
  - Contact Information Cards: Subtle box-shadow on hover

- **Load Animations:**
  - Light fade-in animations for content as it loads

## Image Requirements
1. **Section Header Image:**
   - Description: Abstract, futuristic visual with a human element, showing a scientist in a lab setting to convey innovation and high-tech capabilities.
   - Usage: Background image for the section header.

2. **Map Image:**
   - Description: An interactive map with a pin on the location of BKF Pharma’s main office.
   - Usage: Embedded map showing the location.

## Content Requirements

   - Title: "Get in Touch"

   - **Phone Numbers:**

     - Partner Relations: +1 (800) 234-5678
   - **Physical Address:**
     - BKF Pharma Inc.
     - 123 Innovation Drive
     - Science City, CA 94000

3. **Contact Form:**
   - **Fields:**
     - Name
     - Email
     - Subject
     - Message
   - **Submit Button Text:**
     - Send Message

By adhering to these guidelines, the design team can create a cohesive, visually appealing, and functional Contact Information section for the BKF Pharma website that effectively facilitates communication and engagement with potential partners and stakeholders.
